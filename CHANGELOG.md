# Changelog

## 0.1.1 - 2026-05-23

- Fixed bundled default texture URLs so Vite consumer builds emit the WebP assets into production output.

## 0.1.0 - 2026-05-18

- Initial release of `react-earth-lite`.
- Added a lightweight React Three Fiber Earth component with NASA-derived day, cloud, night, normal, and specular textures.
- Added configurable radius, rotation speed, texture paths, quality mode, reduced-motion support, optional atmosphere, optional ocean specular, and instanced markers.
- Published as an ESM package with bundled WebP texture assets and TypeScript declarations.
